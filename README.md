## Description

This repository contains the source code for Muhamad Fadlan Anshari's personal portfolio website. The website is designed to be a dynamic and interactive showcase of his web development projects, professional experience, and technical skills. It leverages modern frontend technologies to provide a smooth user experience.

## Key Features

*   **Responsive Design**: Adapts to various screen sizes for optimal viewing on desktops, tablets, and mobile devices.
*   **Interactive Components**: Features smooth scrolling, animations, and hover effects to enhance user engagement.
*   **Project Showcase**: Displays a curated list of completed projects with descriptions and links.
*   **Experience Timeline**: Details educational background and work experience.
*   **Contact Information**: Provides easy ways to get in touch via LinkedIn, GitHub, and email.

## Tech Stack

*   **Frontend Framework**: React
*   **Build Tool**: Vite
*   **Styling**: Tailwind CSS, CSS
*   **UI Libraries/Components**: Flowbite React, Hover.css
*   **Animations**: AOS (Animate On Scroll), Tailwind CSS Animations
*   **Icons**: React Icons
*   **Deployment**: (Assumed to be deployed on a static hosting service)

## Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/fadlananshari/fadlansporto.git
    cd fadlansporto
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
